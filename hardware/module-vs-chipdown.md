# MDBT42Q 모듈 vs nRF52832 칩다운 — 크기·양산비용 검토

## 최종 결정 (2026-07-10): **칩다운 채택**

사용자 결정 — 무선 인증(의도적 방사체)을 어차피 받을 계획이라 모듈의 인증 승계 이점이 무의미하고,
단가($2.7 vs $3.5+)와 실장 유연성에서 칩다운이 유리. 회로도는 nRF52832-QFAA +
32 MHz 수정 + Johanson 2450FM07A0029(nRF52 전용 매칭 LPF) + 2450AT18B0100001E 칩안테나로 갱신됨.
전제 조건: RF 인증 예산(KC/FCC/CE RED)과 안테나 튜닝(VNA) 리소스 확보. 아래는 결정 당시의 비교 분석 원문.

---

(참고) 원래 분석 결론: 프로토타입~연 1만 대까지는 모듈 유지가 정석 — 크기는 코일·배터리가 결정하고,
칩다운의 단가 절감은 인증·RF 개발비(NRE)를 1만 대 이상 팔아야 회수한다.

## 1. 최종 크기는 무엇이 결정하나

"표준 Qi 무선충전 + LIR2032" 조합을 유지하는 한, 바꿀 수 없는 두 부품이 크기를 못박는다:

| 부품 | 크기 | 비고 |
|---|---|---|
| Qi RX 코일 (TDK WR202010-18M8-ID) | **20×20×1.0 mm** | 저부하(0.13 W)라 소형 코일 가능 |
| LIR2032 + 홀더 | **Ø20 mm, 높이 ~4.5 mm** | 셀 규격 고정 |
| → 디바이스 외형 | **~22×22 mm 사각 퍽, 두께 ~10 mm** | 크기 하한 = 배터리(Ø20)·코일(20×20) — 최종 보드는 배선 여유 포함 **26×24 mm** |

Ø23 mm 원형 PCB의 실장 면적은 한 면 ~415 mm², 양면 ~830 mm².

| MCU 구현 | 점유 면적 | PCB 양면 대비 |
|---|---|---|
| MDBT42Q 모듈 | 160 mm² (16×10) | **19 %** |
| nRF52832 칩다운 (QFN48 6×6 + 32 MHz 크리스탈 + 칩안테나 + **안테나 킵아웃 ~55 mm²** + RF 수동소자 ~12개) | ~110 mm² | 13 % |

**절감분은 ~50 mm² = PCB의 6 %.** 외형은 배터리(Ø20)가 정하므로 칩다운해도
디바이스는 작아지지 않는다 (Ø23 보드에 모듈 19 % 점유 — 배치 여유 충분). 높이도 모듈(2.2 mm) < 셀 홀더(~4.5 mm)라 무관.

## 2. 배치 계획 (Ø23 mm 원형 PCB 기준)

```
        [12시] MDBT42Q — 안테나가 보드 가장자리 밖을 향하게, 킵아웃 확보
   [9시] 전원부(BQ51013B·탱크 캡·MCP73832)     [3시] MIC5205 + 디커플링
        — 뒷면 코일 인출선과 최단 연결
   [6시] AFE (REF35102·OPA2391·RF/CF) + 전극 패드/커넥터
        — 코일·스위칭 노드에서 최대 이격, WE 노드 가드링
   뒷면: LIR2032 홀더(중앙), SWD 패드, VBAT 분압
   보드 아래: Qi 코일(20×20×1.0) 적층 → 총 두께 ~10 mm
```

빡빡하지 않다. 모듈을 써도 AFE를 노이즈원에서 충분히 떼어놓을 여유가 있다.

## 3. 양산 비용 비교 (1k 기준 개략)

| 항목 | MDBT42Q | 칩다운 |
|---|---|---|
| 부품 단가 | 모듈 ~$3.5 | nRF52832 ~$2.3 + 크리스탈/안테나/수동소자 ~$0.4 = **~$2.7** |
| PCB | 2층 가능 (RF 없음) | RF 임피던스 컨트롤 필요 → 4층/공차 관리 +$0.1~0.3 |
| **단가 차** | — | **개당 ~$0.8~1.0 절감** |
| RF 설계 NRE | 0 | 안테나 매칭·튜닝 (챔버/외주) $3k~8k |
| 무선 인증 | **모듈 인증 승계** (KC·FCC·CE 모듈러) → 완제품 간이시험 ~$1k~3k | **의도적 방사체 신규 인증** KC + FCC + CE ~$10k~20k |
| 설계 리스크 | 없음 (검증된 RF) | 첫 스핀에 안테나 성능 실패 리스크 → 리스핀 비용·일정 |

**손익분기 ≈ (NRE $13k~25k) ÷ (개당 $0.8~1.0) ≈ 1.5만~2.5만 대.**

## 4. 결론과 전환 기준

- **지금~파일럿 양산: MDBT42Q-512KV2 유지.** 크기 이득 0, 확실성·인증·일정 이득 큼.
- **칩다운 전환을 검토할 시점** (아래 중 하나면):
  1. 연 판매 계획이 **2만 대 이상**으로 잡힐 때
  2. 표준 Qi를 포기하고(코일 소형화/독자 충전) **디바이스 지름을 <20 mm로 줄이는 재설계**를 할 때
     — 이 경우 어차피 전체 리레이아웃이므로 그때 칩다운을 같이 하는 게 맞다
- 전환 시에도 이 회로도의 **AFE·충전·전원 블록은 그대로 재사용** — 바뀌는 건 U3 블록(모듈 → 칩+RF)뿐.
- 중간 단계 대안: 더 작은 모듈(ANNA-B112, 6.5×6.5 mm)로 교체하면 재인증 없이 실장 면적을
  1/4로 — 단, 이것도 최종 외형은 안 줄어들기는 마찬가지.
