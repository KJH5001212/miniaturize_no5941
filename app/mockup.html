Ampero — 웨어러블 포텐쇼스탯 전용 앱 목업 v1.1

CGM 문법으로 직관성을 끌어올린 라운드. 값·그래프를 구간 색으로 말하고, 추세 화살표·오전/오후 TIR 시계·주간 패턴·튀는 값 삭제까지. 표기는 전류(nA).

저전류 <80 안전 80–240 고전류 >240

v0.9 변경 (요청 8건)

  • 구간 색상: 안전=초록·저전류=빨강·고전류=주황. 그래프 배경 3존 음영 + 곡선 구간별 색 + 현재값·화살표 색
  • 추세 화살표: 위/평형/아래 3단계 (최근 30분 기울기), 숫자 옆
  • 오전·오후 시계 2개: 꽉 찬 파이(구간색), 미래 시간대는 빈 파이(테두리만). 빨강·주황 특이점엔 탭 마커(!) → 누르면 홈 그래프가 그 시간대로 점프. 정상 비율 %는 파이 아래 라벨로 이동
  • 홈 위젯: 플랫폼 API 필요 — 메모만 (아래 백로그)
  • 기록 일간: 날짜를 탭하면 월·일 스크롤 휠로 이동. 주간 패턴 탭 = 7일 겹쳐 중앙값+10–90% 밴드(AGP)로 반복 패턴 확인
  • 알림: 설정에서 켜고 끄는 토글만 (동작은 앱 빌드 시)
  • 튀는 값 삭제: 점을 탭하면 하단 바에 값+삭제 버튼 → 곡선·통계에서 제외

백로그 (나중)

  • 홈 위젯 / 워치 컴플리케이션 (현재값+화살표 글랜스)
  • 임계 알림 실제 푸시 (백그라운드 동작 실기 테스트)
  • 삭제한 값 서버 동기화(soft-delete) · 되돌리기
