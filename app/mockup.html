Ampero — 웨어러블 포텐쇼스탯 전용 앱 목업 v0.1

상용 CGM(연속혈당) 앱 문법 그대로, 표기만 전류(nA). 왼쪽 폰에서 탭·그래프·설정을 직접 눌러볼 수 있어. 이 화면 구성이 확정되면 Kotlin(Compose)으로 실제 앱을 빌드한다.

펌웨어 ↔ UI 매핑

d[[…]]홈 판독값·스파크라인·그래프 (agg 모드 = 사이클당 1점)
vbat상단 배터리 % (전압→% 룩업), 설정→기기
qi + chg충전 배너: 패드 밖 / 충전 중 / 완충
fpend홈 상단 동기화 배너 + 진행률
gap기록 탭 "데이터 유실 구간" 경고
lb저전압 경고 배너 + 측정 시작 차단 안내
rf / os / agg설정 → 하드웨어·측정 (config 명령 전송)
zero / cal설정 → 캘리브레이션 버튼 2개
