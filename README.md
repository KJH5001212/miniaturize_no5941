# Discrete Potentiostat (초소형 무선충전 크로노암페로메트리)

AD5941 없이 디스크리트 듀얼 앰프로 구성한 3전극 크로노암페로메트리 디바이스.
WE-RE 간 **+0.5 V 고정 바이어스**, nA~µA 전류 측정, BLE 스트리밍, Qi 무선충전.

## 확정 부품

| 블록 | 부품 | 역할 |
|---|---|---|
| AFE | **OPA2391** (듀얼) | A: 컨트롤 앰프 (CE 구동, RE 센스) / B: TIA (WE 전류→전압) |
| 레퍼런스 | **REF35102QDBVR** (1.024 V) | WE 가상전위 + 바이어스 분압 원천 |
| MCU/BLE/ADC | **nRF52832** | 내장 SAADC(차동, 12-bit+오버샘플링)로 측정, BLE 전송 |
| LDO | **MIC5205-3.3** | 3.3 V 단일 레일 |
| 무선충전 | **BQ51050BRHLR** | Qi 수신 + 리튬 충전 일체형 (4.2 V, LIR2032 직결) |
| 배터리 | **LIR2032** | 3.7 V / ~40 mAh 코인셀 |

## 신호 체인

```
Qi 코일 ─ BQ51050B ─ LIR2032(4.2V) ─ MIC5205 ─ 3.3V ─┬─ nRF52832
                                                      └─(GPIO 스위치드)─ REF35102, OPA2391

REF35102 1.024V ──────────────────────────── TIA(+in): WE 가상전위 = 1.024V
      └─ 분압(1.00M/1.00M, 1:1) → 0.512V ── 컨트롤앰프(+in): RE 목표

E_cell = WE − RE = 1.024 − 0.512 = +0.512 V (대략 0.5 V 고정, 요구사항 충족)

WE → OPA2391-B (TIA, RF=1MΩ) → V_out = 1.024 + I×RF
V_out, 1.024V → nRF52832 SAADC 차동 입력 → I = ΔV / RF → BLE
```

## 기존 ad5941 프로젝트와의 관계

BLE 프로토콜(NUS + JSON), 무손실 버퍼링, 런 모드(연속/타이머/사이클), Android 앱은
**그대로 재사용**한다. 교체되는 것은 측정 엔진뿐: SPI+AD5941 → SAADC+디스크리트 AFE.
